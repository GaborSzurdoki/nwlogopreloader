== Description ==

Creates JavaScript-powered custom-branded preloader for your WordPress page with CSS3 animations. You simply upload your logo, set the animation type, and have a well-designed preloader on all of your pages. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/nw-logo-preloader` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Appearance->NW Logo Preloader to configure the plugin
4. Enable it, upload your logo (128x128 or 256x256 pixel transparent PNG preferred)
